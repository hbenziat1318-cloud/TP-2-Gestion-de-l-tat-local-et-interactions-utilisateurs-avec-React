
#  TP React - Gestion d'État Local et Interactions Utilisateurs


## Objectifs du TP

### Objectifs Pédagogiques
- **Maîtriser** la gestion d'état local avec `useState`
- **Comprendre** le cycle de vie des composants avec `useEffect`
- **Implémenter** des interactions utilisateur avancées
- **Créer** des formulaires contrôlés
- **Appliquer** le flux unidirectionnel des données

### Compétences Validées
| Compétence | Statut | Niveau |
|------------|--------|---------|
| Gestion d'état avec useState |  Acquise | Avancé |
| Effets secondaires avec useEffect |  Acquise | Intermédiaire |
| Formulaires contrôlés |  Acquise | Avancé |
| Gestion d'événements |  Acquise | Avancé |
| Design responsive |  Acquise | Intermédiaire |

##  Fonctionnalités

###  Composants Interactifs

| Composant | Icône | Description | Fonctionnalité Principale |
|-----------|-------|-------------|---------------------------|
| **Compteur Simple** | Incrémentation basique | `useState` pour le comptage |
| **Message Bouton**  | Interaction texte | Changement dynamique |
| **Formulaire Nom**  | Saisie unique | Formulaire contrôlé |
| **Compteur Effet**  | Logging des changes | `useEffect` avec console |
| **Affichage Dynamique** | Messages progressifs | Tableau de messages |
| **Formulaire Inscription**| Multi-champs | Gestion de plusieurs états |
| **Compteur Titre**| Synchronisation | Titre de page dynamique |
